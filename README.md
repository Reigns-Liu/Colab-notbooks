## How to Use  

### Prerequisites  
- A web browser and internet access  
- (Optional) Google Account to sign into Colab  
- Basic familiarity with Python and notebooks  

### Running a Notebook  
1. Navigate to the notebook file in this repository (e.g., `*.ipynb`).  
2. Click the **‚ÄúOpen in Colab‚Äù** or similar badge/link (if provided), or go to `https://colab.research.google.com/github/Reigns-Liu/Colab-notbooks/blob/master/<path-to-notebook>.ipynb` to open it in Colab.  
3. Once opened, you can execute the cells, modify code, and explore the workflow.  
4. To save your changes, you can use File ‚Üí Save a copy in GitHub or save to Google Drive.  

### Contributing Your Own Notebook  
- Fork the repository.  
- Add your notebook file under an appropriate folder (e.g., `notebooks/`).  
- Ensure it is well-documented (with headings, comments, and dataset references).  
- Submit a Pull Request with a clear description of what the notebook covers and how to run it.
